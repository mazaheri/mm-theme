# MM Theme

WordPress theme for Mazz Marketing.

## Current status

Every URL (`front-page.php`, `page.php`, `index.php`) renders the Coming Soon
launch screen. Copy, social links, and colors are editable via
**Appearance → Customize → Coming Soon Page** (see `mm_theme_customize_register()`
in `functions.php`).

## Next steps

Additional page templates (home, blog, services, etc.) will replace the
Coming Soon fallback as designs are finalized. Follow the theme bible in
`version 4 download image and blog/` and `version 5 import demo workflow/`
for the conversion, dynamic-layer, and deploy methodology used on this project.
